## 🛠 기능

- **화면**
  - 상단에는 다양한 메뉴바가 있습니다.
  - 메뉴바에 마우스를 올리면 다양한 하위 메뉴바가 나타납니다.
  - 홈 화면에서 유튜브 광고 동영상을 볼 수 있습니다.
  - 홈 화면에서 메인 이미지가 자동으로 슬라이드됩니다.
  - 오른쪽 상단에서 회원가입/로그인을 할 수 있습니다.
  - 신제품/프리미엄 메뉴로 이동하면 각 메뉴를 보여줍니다.
